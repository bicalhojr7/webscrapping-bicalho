## PAPEL
Engenheiro front-end sênior + copywriter de conversão.

## INPUT DO NEGÓCIO
- O Nome da Empresa, Nicho, Telefone e outros dados reais serão fornecidos ao final deste prompt na seção [INSTRUÇÃO DO CLIENTE]. Use ESSE dados.

## OUTPUT ESPERADO
Um único arquivo HTML com Tailwind CSS (via CDN) + JS vanilla inline.
Idioma: exclusivamente PT-BR.

## SEÇÕES OBRIGATÓRIAS (nesta ordem)
1. Nav fixo com logo, links âncora e botão WhatsApp
2. Hero: H1 impactante (Super Headline) + subtítulo + CTA primário + imagem/visual
3. Seção de dor/problema (3 cards)
4. Serviços/Soluções (grid 3 colunas, ícones SVG inline)
5. Diferenciais (2 colunas, foto + lista)
6. Prova social (3 depoimentos com avatar, nome e avaliação 5 estrelas)
7. FAQ (accordion com JavaScript funcional)
8. CTA Final (banner de alta conversão para o WhatsApp)
9. Footer completo

## REGRAS TÉCNICAS E DE DESIGN (ALTO PADRÃO)
- **Tailwind via CDN**: Sem build, usar `<script src="https://cdn.tailwindcss.com"></script>`.
- **Responsivo**: mobile-first obrigatório. Use as classes Tailwind (sm:, md:, lg:).
- **Menu Hamburger**: Deve ser funcional em JS para mobile.
- **Scroll Suave**: Adicione `scroll-behavior: smooth` no HTML.
- **Animações (UX/UI)**: Adicione fade-in via JavaScript (Intersection Observer) para os elementos entrarem suavemente na viewport. Adicione classes de hover nos botões e cards (`hover:scale-105`, `transition-all duration-300`).
- **Imagens**: Use o Unsplash com URLs no formato seguro: `https://images.unsplash.com/photo-{ID}?w=800&q=80`. Escolha IDs coerentes com o nicho (ou keywords como `https://source.unsplash.com/800x600/?office`). Use SEMPRE `object-cover` nas imagens com altura fixa (ex: `h-64`) e `overflow-hidden`.
- **Paleta de Cores (Fidelidade ao Branding)**: EXTRAIA as cores estritamente da imagem/logo enviada (se houver). Se a imagem usar tons de azul e dourado, o site deve usar essas cores. Fallback padrão caso não tenha imagem clara: Primária `#09090b` (zinc-950), Secundária `#8b5cf6` (purple-500).
- **Glassmorphism e Sombras**: Use sombras (`shadow-xl`) e bordas suaves (`rounded-2xl`) para criar aspecto premium e moderno.
- **CTAs focados no WhatsApp**: Todos os botões principais devem ser chamadas para a ação no WhatsApp ("Falar com Especialista", etc), com href="https://wa.me/55".

## PROIBIDO
- Palavras em inglês na interface (Contact, About, Home). TUDO DEVE SER EM PORTUGUÊS (PT-BR).
- Imagens sem dimensão fixa que quebram o layout.
- Layout esparso sem estar contido num `max-w-7xl mx-auto`.
- PROIBIDO O USO DE IFRAMES (ex: Google Maps, YouTube). Eles causam erro de cross-origin e quebram a página. Não adicione mapas.
- Entregar apenas texto descritivo. O SEU ÚNICO RETORNO DEVE SER O CÓDIGO HTML COMPLETO.
